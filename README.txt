api виртуальная книжная библиотека

использовались openserver и phpmyadmin

настройки open server:
Apache_2.4+PHP_8.0+Nginx_1.21
PHP_8.0
MySQL-8.0-Win10

база данных лежит в файле library.sql

для добавления записи в таблицу, нужно использовать команду
http://apilibrary/take/101/1 - где 101 это номер читательского билета пользователя, а 1 номер книги (id)
диапазон пользователей 101-104 (таблица user), книг 1-7 (таблица book)

для возвращения книги нужно использовать
http://apilibrary/return/101/1 - где 101 это номер читательского билета пользователя, а 1 номер книги

для просмотра книг у определенного читателя -
http://apilibrary/view/101 - где 101 это номер читательского билета
